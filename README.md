## 🚀 How It Works — Step by Step

### Method 1: The Magic Auto-Scan (Zero Prompt)

```
Step 1  →  Open any project folder in VS Code
Step 2  →  Press Ctrl+Shift+P (or Cmd+Shift+P on Mac)
Step 3  →  Type "Diagram Weaver: Weave Diagram" and press Enter
Step 4  →  Leave the prompt box EMPTY and hit Enter
Step 5  →  Watch your entire architecture appear in seconds ✨
```

**What happens under the hood:**
```
Your Project
     │
     ▼
📂 Deep File Scanner ──► reads package.json, imports, folder structure
     │
     ▼
🧠 AI Model Engine ──► GPT-4o / Claude 3.5 / Copilot (auto-selected)
     │
     ▼
🎨 Dark-Mode Renderer ──► Mermaid.js with handDrawn look
     │
     ▼
🖼️ Beautiful Diagram ──► Rendered in VS Code WebView panel
```

### Method 3: Use It From Copilot Chat

```
Step 1  →  Open Copilot Chat panel in VS Code

Step 3  →  Your diagram renders instantly in a new panel
```

**Available Copilot Chat commands:**
| Command | What It Does |
|---------|-------------|

## ✨ Core Features

### 🔍 Zero-Prompt Auto-Architect Engine
Diagram Weaver **deep-scans your workspace** automatically — reads your `package.json`, dependency trees, folder structure, and import chains. It builds a complete high-level architecture diagram of your project **without you typing a single word.**

### 🎨 Hand-Drawn Dark-Mode Aesthetic
Every diagram is rendered with a premium **hand-drawn look** on a deep dark canvas (`#0A0A0A`). It looks exactly like the diagrams you see on Eraser.io — but generated for free, inside your editor.

### 🤖 Smart AI Model Auto-Selection
No API key required. Diagram Weaver hooks directly into **VS Code's native Language Model API** (the same engine that powers GitHub Copilot). It automatically selects the most powerful available model (`GPT-4o`, `o1`, `Claude 3.5`) to ensure maximum intelligence for complex diagrams.

### 📤 One-Click Export
Export your diagram as a **pixel-perfect PNG** or a **mathematically preserved SVG** directly from the toolbar inside the VS Code panel. Perfect for documentation, Notion pages, or Confluence.
